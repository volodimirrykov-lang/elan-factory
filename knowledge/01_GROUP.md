## Как связаны (MFJ — Material Financial Journey)

```
                        [Собственники: АГ + ТГ]
                                  │
         ┌───────────────┬────────┴─────────┬─────────────────┐
         │               │                  │                 │
    ELAN KİMYA      ФОП Кравченко      ELAN BEAUTY      ELAN COSMETICS
    (TR)            (UA)               (PL)             (UAE)
         │               │                  │                 │
         │        ┌──── sales ───┐          │                 │
         └───────►│              │◄─────────┴─────────────────┘
          export │              │
          invoices│              │   (от всех трёх — платежи в TR)
                  │              │
                  ▼              ▼
            Внешние клиенты (LUNESI UK, USUPSO, Cosmoprof, Aramark...)
```

**Суть:** TR завод производит товар, продаёт через:
- Часть напрямую на экспорт (LUNESI UK — 16% марта)
- Часть через ФОП UA (20% марта) → ФОП продаёт дальше по Украине + выставляет счета TR-клиентам
- Часть через PL-компанию (16% марта) → EU растаможка, дальше европейским клиентам
- Часть через UAE (2% марта) → MENA канал

## Юридические риски связанных сторон

- **Transfer pricing**: сколько TR продаёт ФОПу — по каким ценам? Если слишком низко — налоговая TR накроет. Если слишком высоко — налоговая UA накроет
- **Dividend repatriation**: ТГ и АГ получают средства как? Без loan agreement — для DD мутно
- **Substance в UAE**: ELAN COSMETICS в Дубаи должна иметь реальную substance (офис, сотрудник), иначе не зачтётся
- **Sanctions screening**: UA клиенты — проверка на санкционные списки (мелочь, но инвестор спросит)
